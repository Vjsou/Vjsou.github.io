Jeweled

Download this project as a .zip file for Windows
Download this project as a .zip file for Android

How to run the game

Windows: Download and extract the .zip archive. Run Jeweled.exe.

Android: Download and extract the .zip archive. Run the Install_Jeweled-armv7 with your android device connected to the PC.

Description

Currently a prototype, Jeweled is a personal project. The goal is to score a maximum of points in a fixed number of action.

You can also save your score and your pseudo at the end of each game

Keys

  • Right mouse click on an Element then on one of his neighbours to swap them
  • You can also hold right mouse click on an Element then release on one of his neighbours to swap
